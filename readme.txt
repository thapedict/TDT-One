A simple retro theme with a serif font face to for those who prefer content over looks.

----------

== Description ==

This was designed with reading in mind and as such is very easy on the eye with a retro look and feel. 

The things you can customize:
1. Main colour
2. Main Shade Background colour
3. Footer text colour
4. Footer background colour
5. Footer strip text colour
6. Custom logo
6. Social media links
